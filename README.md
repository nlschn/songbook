# Usage (Local Debug Mode)
1. Set up and activate virtual environment with requirements
2. Install redis
3. Prepare your Spotify account with API access
4. Set environment variables `SPOTIPY_CLIENT_ID` and `SPOTIPY_CLIENT_SECRET` with your according API credentials
5. Set environment variables `FLASK_APP` to `songbook` and `FLASK_DEBUG` to `true`
6. `flask run`
